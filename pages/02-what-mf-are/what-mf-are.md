# What are Microfrontends?
## Core concepts
<v-clicks>

* Independent deployment
* Incremental upgrades
* Autonomous teams
* Decoupled codebases
* Technology agnostic
* Compile time integration vs Runtime integration
 
</v-clicks>

# What are Microfrontends?
## Downsides
<v-clicks>

* Payload size
* Environment differences (prod vs dev)
* Operational complexity
* Inconsistent UI
 
</v-clicks>
